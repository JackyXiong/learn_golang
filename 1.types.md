# 类型
值类型：变量名存的是值
引用类型：变量名存的是地址

#### 基本类型

| 类型           | 长度（字节） | 默认值   | 说明                       |
| ------------ | ------ | :---- | ------------------------ |
| bool         | 1      | false |                          |
| byte         | 1      | 0     |                          |
| rune         | 4      | 0     |                          |
| int,unit     | 4/8    | 0     |                          |
| int8,unit8   | 1      | 0     | -128~127， 0~255 (2的8次方)  |
| int16,unit16 | 2      | 0     | -128~127， 0~255 (2的16次方) |
| int32,unit32 | 4      | 0     | -21亿~21亿， 0~42亿 (2的32次方) |
| int64,unit64 | 8      | 0     |                          |
| float32      | 4      | 0.0   |                          |
| float64      | 8      | 0.0   |                          |
| complex64    |        |       |                          |
| complex128   |        |       |                          |
| array        |        |       | 值类型                      |
| struct       |        |       | 值类型                      |
| string       |        | ""    | utf-8字符串                 |
| slice        |        | nil   | 引用类型                     |
| map          |        | nil   | 引用类型                     |
| channel      |        | nil   | 引用类型                     |
| interface    |        | nil   | 接口                       |
| function     |        | nil   | 函数                       |
